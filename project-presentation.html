üîå Reporting Framework

Plugin-based TypeScript library for LLM-powered report generation

Core Concepts

  • Plugin: Self-contained modules for data ingestion & processing
  • Bundle: Standardized container for data (FHIR-inspired)
  • Specification: YAML config defining report structure
  • Template: MDX or Markdown for final rendering
  • Renderer: Converts output to HTML, PDF, or other formats

Technology Stack

  • Language: TypeScript 5 (100% type-safe)
  • Template Engine: Nunjucks + MDX
  • React Integration: React 18 & 19 compatible
  • LLM Support: Vercel AI SDK
  • PDF Generation: Playwright + Chromium
  • Component System: Built-in MDX components
  • Package Manager: npm/pnpm compatible

Quick Start Example

import { PluginRegistry, ReportEngine, MDXRenderer } from "@aganitha/reporting-framework";

const registry = new PluginRegistry();
registry.register(new MyPlugin());

const engine = new ReportEngine(registry);
const bundle = await registry.getPlugin("my-plugin").ingestFromFile("data.json");

const report = await engine.generateReport({
  pluginId: "my-plugin",
  specificationId: "default",
  bundle,
  llmConfig: { provider: "openai", model: "gpt-4" }
});

const mdxRenderer = new MDXRenderer();
const html = await mdxRenderer.render(report);
